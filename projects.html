Full-stack automation

Ama Events Co.

A booking site and AI-powered lead-triage automation for an event planning, decor, and rental business. The business problem: inquiries scattered across WhatsApp, Instagram, Facebook comments, and a website form, with one overloaded owner trying to triage all of it manually. The system reads every inbound message, classifies it by urgency and category, replies appropriately on whatever channel it came in on, logs it to Airtable, and alerts the owner only when something actually needs her attention.

n8n AI Agents (Claude / Groq) Airtable Meta Graph API Gmail Telegram Vercel

What made this a real build, not a demo

This went through an actual audit before being called done — not a happy-path showcase. Some of what surfaced:

  • A Form Trigger vs. Webhook mismatch that would have silently broken every site submission
  • A completely empty WhatsApp reply node — would have fired blank requests with zero indication anything was wrong
  • A placeholder-syntax mismatch that could have leaked raw text to real customers instead of a working link
  • A returning-customer detection bug caused by a date-only timestamp — fixed using full-precision metadata instead
  • A cross-provider email deliverability gap: reliable to Gmail, silently dropped by 2 of 3 Outlook test addresses — traced to sender reputation, not code
